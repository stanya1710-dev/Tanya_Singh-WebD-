# Tanya_Singh-WebD-
# Multi-Page Student Portfolio Website
## Project Overview
A simple 4-page personal portfolio website built using only **HTML5** and **Basic CSS**. Demonstrates navigation, layouts, forms, and responsive design principles.
## Pages Included
- **index.html** - Home page with welcome message
- **about.html** - About me page with profile image and introduction
- **skills.html** - Skills & Projects page with 6 skill cards (Flexbox layout)
- **contact.html** - Contact form page with input validation styling
## Key Features Demonstrated
- ✅ **Navigation Bar** (Float layout) - Common navbar across all pages
- ✅ **Flexbox Layout** - Skills cards alignment
- ✅ **Float Layout** - Logo and menu positioning
- ✅ **HTML Forms** - Fully styled contact form
- ✅ **Semantic HTML** - Proper structure with headings, sections
- ✅ **CSS Styling** - Colors, borders, spacing, hover effects
## Technologies Used
- **HTML5** - Structure and semantic markup
- **CSS3** - Styling, Flexbox, Float layouts
- **Git & GitHub** - Version control and hosting
